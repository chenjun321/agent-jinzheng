# AI 使用说明

## 1. 使用原则

本项目允许并鼓励使用 AI 编程工具，但需要说明：

- AI 用在了哪些环节。
- 哪些结果经过人工校验。
- 哪些地方进行了修正。
- 没有把 AI 输出作为不可解释的最终结果直接提交。

## 2. 计划使用方式

AI 辅助范围：

- 技术方案拆解。
- 项目结构设计。
- 代码骨架生成。
- README 和设计文档初稿。
- 测试用例设计。
- 边界情况梳理。

人工负责范围：

- 校验 OCR 结果。
- 检查表格提取是否合理。
- 检查检索证据是否支持答案。
- 调整无答案拒答阈值。
- 运行测试和评估脚本。
- 确认 README 与实际功能一致。

## 3. 禁止做法

本项目不会：

- 手工录入 PDF 全文绕过解析。
- 为演示问题硬编码答案。
- 提交 API Key 或账号密码。
- 将无法解释的 AI 输出直接作为最终结果。
