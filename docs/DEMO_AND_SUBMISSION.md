# 演示与提交说明

## 1. 提交内容

最终提交应包含：

- GitHub 或 Gitee 仓库链接。
- README。
- 设计说明。
- 测试说明。
- 5-10 分钟演示视频或清晰截图。

## 2. 演示流程建议

演示视频按以下顺序展示：

1. 项目目录和 README。
2. 本地安装依赖。
3. 执行 PDF 解析。
4. 查看解析结果，包括正文和表格。
5. 执行至少 5 个问题问答。
6. 展示表格问题结果。
7. 展示无答案问题拒答。
8. 展示来源引用和自检结果。
9. 运行评估脚本。
10. 简要说明 AI 使用和人工校验。

## 3. 截图清单

如果使用截图替代视频，建议准备：

- 安装和启动截图。
- PDF 类型检测截图。
- OCR 页面结果截图。
- 表格提取结果截图。
- 普通正文问答截图。
- 表格问答截图。
- 无答案拒答截图。
- 评估脚本结果截图。

建议格式：

```text

```
